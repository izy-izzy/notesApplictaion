addNoteController
controller in module notesApp

Description

enables adding new note

Dependencies

  • SweetAlert
  • databaseService
  • uidFactory
  • authService

Methods

  • addNewNote()

    Adds a new note. Checks whether text and title is not empty and inserts a note. If text or title is empty user, is warned and action is canceled. New note is added with current time, title, active user id and comment (which includes text, current time and active user's id) to this note.
