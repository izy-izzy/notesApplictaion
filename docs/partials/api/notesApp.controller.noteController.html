Methods

  • addNewComment()

    Adds a new comment. Checks whether text is not empty and inserts the comment to the active note. If text is empty, user is warned and action is canceled. New comment is added with text, current time and userId of the active user.

  • checkNoteStatus()

    Checks if the current note is empty. If the note is empty/deleted, warning is shown and user is rerouted to notes overview.

  • deleteComment(id)

    Deletes a comment. Prompts to confirm request and according to selected action it deletes the comment or cancel the request.
