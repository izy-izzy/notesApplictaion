customDateFilter
service in module notesApp

Description

Modifies date to a specific string, eg. 'Today at 12:35', 'Yesterday at 12:35', '15/1/1988 at 12:35'.

Usage

In HTML Template Binding

{{ customDateFilter_expression | customDateFilter }}

In JavaScript

$filter('customDateFilter')(date)

Parameters

ParamTypeDetails
datenumber

in ms from start of 1970

Returns

string

time with modified format
